Project Saya

Xplore – Website Eksplorasi Wisata

Xplore adalah website layanan wisata alam yang dibangun menggunakan HTML, JavaScript, dan Bootstrap. Website ini menyediakan informasi serta layanan pendampingan bagi pengunjung yang ingin menikmati pengalaman wisata alam secara aman, seru, dan terorganisir.

Jadoo – Website Layanan Wisata dan Pemesanan Perjalanan

Jadoo adalah website pemesanan wisata berbasis HTML, CSS, PHP, dan MySQL. Dirancang responsif dengan fitur informasi destinasi dan sistem booking perjalanan yang terhubung ke database.

Rental Buku – Program Manajemen Penyewaan Buku (C++)

Program manajemen penyewaan buku berbasis C++ dengan fitur peminjaman, pengembalian, perhitungan biaya, dan penyimpanan data menggunakan file handling serta struktur data Linked List dan Stack.

Analisis Data Harga Kripto Menggunakan Python dan Excel

Analisis perbandingan harga aset kripto menggunakan Excel dan Python di Google Colab. Proyek ini memanfaatkan Pandas dan Matplotlib untuk pengolahan serta visualisasi data harga kripto secara terstruktur.

Sistem Manajemen Perpustakaan (Java)

Program manajemen perpustakaan berbasis Java yang mengimplementasikan konsep Object Based Programming seperti inheritance, encapsulation, dan polymorphism untuk mengelola data buku, anggota, serta transaksi peminjaman.
